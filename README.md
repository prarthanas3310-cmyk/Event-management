# 📅 Event Management System

A full-stack **Event Management & Registration System** built using **MongoDB, Express.js, React, and Node.js (MERN Stack)**.

The application provides secure user authentication, event management, attendee registration, and a responsive user interface with dark/light mode support.

---

# 🚀 Features

## 🔐 Authentication
- User Registration
- User Login
- Password Hashing using Bcrypt
- JWT Authentication

## 📅 Event Management
- Create Events
- View Events
- Edit Events
- Delete Events

## 🎟️ Registration Management
- Register Participants
- View Registration Details
- Edit Registration
- Delete Registration

## 🌗 User Interface
- Responsive Design
- Dark & Light Mode
- Clean Dashboard

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/prarthanas3310-cmyk/Event-management.git
cd Event-management
```

## Backend Setup

```bash
cd backend
npm install
npm start
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
