# 📅 Event Management System

A full-stack Event Management & Registration System built using **React, Node.js, Express, and MongoDB**.  
It provides secure authentication, event creation, and attendee registration management.

---

## 🚀 Features

### 🔐 Authentication System
- User registration & login  
- Password hashing (bcrypt)  
- JWT-based authentication  

### 📅 Event Management
- Create new events  
- View all events  
- Update event details  
- Delete events  

### 🎟️ Registration System
- Register users for events  
- Track attendees per event  
- Manage registration data  

### 🌗 UI Features
- Dark / Light mode support  
- Responsive design  
- Clean dashboard UI  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- React Router  
- Context API (Theme management)  
- Fetch API  

### Backend
- Node.js  
- Express.js  
- MongoDB  
- JWT Authentication  
- Bcrypt.js
